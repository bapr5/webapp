# Установка зависимостей
- прописать в директории /www `npm i`
- прописать в директории /www/my-react-app `npm i`

# Приложение
- перед использованием приложение нужно СОБРАТЬ. сборка происходит через webpack, собранный js хранится в
./dist/.
- для сборки использовать `npm run build`
- для того, чтобы запустить сервер с приложением запустить `npx ws -d dist`
